SassColor

Swift
public final class SassColor: SassValue

A Sass color value.

Color values represent a point in the sRGB color space. They are defined using either RGB-A or HSL-A parameters but can be accessed as either.

note

Parameter values follow web standards rather the Apple SDK standards, so for example ‘red’ is modelled as an integer in 0…255.
